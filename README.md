# h1

## Приклад використання

```bash
go run main.go -m 53 -a 13 -b 2
```

Результат:

```text
1. Задаємо модуль за допомогою флагу -m.
Значення: m=53, a=13, b=2

2. Розв'язуємо рівняння виду a mod m = x.
13 mod 53 = x
x = 13

3. Розв'язуємо рівняння виду a^b mod m = x.
13^2 mod 53 = x
x = 10

4. Розв'язуємо рівняння виду a*x ≡ b mod m.
13*x = 2 mod 53
x = 45

5. Генеруємо просте число у діапазоні від A до B.
Просте число у діапазоні [24, 35] = 29
```
